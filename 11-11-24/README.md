## วันจันทร์ที่ 11 เดือน พ.ย. 2024

### โจทย์ปัญหา
1. **[เปิด สลับตัวแปร](./swap-var/README.md)** จงเขียนโปรแกรม รับจำนวนเต็ม 1 ตัวโดยต้องรับ 4 หลักจากนั้นทำการตัดหลักให้แต่ละหลักเป็ร A B C D จากนั้นแสดงผลค่าตัวแปรทั้งสี่ตัวที่รับเข้ามา จากนั้นทำการสลับข้อมูลในตัวแปรโดยให้ A สลับกับ D จากนั้น ให้ B สลับกับ C แล้วทำการแสดงผลค่าในตัวแปรอีกครั้ง แสดงออกทางหน้าจอ
(เพิ่มเอง).. ง่ายไปหน่อยขอเพิ่มว่าห้ามใช้ ตัวแปรที่ 3 หรือตัวแปรสำรอง ในขั้นตอนการสลับตัวแปร
2. **[เปิด ติวเตอร์](./tutor/README.md)** จงเขียนโปรแกรมคำนวณคำนวณค่าเรียนพิเศษของ ติวเตอร์คณิตศาสตร์ แห่งหนึ่ง โดย ติวเตอร์แห่งนี้ คิดค่าเรียนเป็นรายชั่วโมง ชั่วโมงละ 450 บาท และจ่ายแบบครั้งค่อครั้ง
3. **[เปิด คำนวณปีเกิด](./since/README.md)** ในการคำนวณอายุนำปีปัจจุบัน - ปีที่เกิด ก็จะทราบอายุโดยประมาณ แต่ถ้านำเดือน ปัจจุบัน - เดือนเกิดก็จะทราบเดือนด้วย
